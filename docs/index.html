CookieSH
Root-first toolkit for GSI tinkerers

What is CookieSH?

CookieSH is a toolkit designed for users who work with GSIs and need deeper access to Android’s internal systems. It focuses on clarity, reliability, and giving advanced users the tools they need to inspect and manage their device.

Features

  • Props inspection and editing
  • Logcat viewer with filtering
  • Partition information viewer
  • Shell command runner
  • Package management tools
  • Device information dashboard
