// features
Everything you need.
Nothing you don't.
Props Editor
Read, edit, add and delete system props via getprop. Search, favorites, export, and PHH superuser presets included.
Shell Runner
Real-time output with root toggle, local command history, copy and clear controls. Execute anything, instantly.
Logcat Viewer
Live feed with search, tag/package/level filters, pause/resume, export, and buffer clear. Full control over your device logs.
Package Manager
System, user, and disabled app filters. Disable, enable, uninstall for user 0, force stop — no adb required.
Partition Viewer
Parse /dev/block/by-name with active slot highlighting, size lookup, and per-partition dump support.
ADB & Network
Toggle ADB over Wi-Fi, set port, view IP, and manage PHH superuser package — all from one screen.
